# FRM-GB-DOC-001 - Formato Maestro de Control Documental de GoBeyond

## Identificacion del documento
- Codigo: `FRM-GB-DOC-001`
- Version: `1.0`
- Estado: `Vigente`
- Fecha de emision: `05 de abril de 2026`
- Area responsable: `Administracion / Gobierno Documental`
- Elaborado por: `Lejzer Trana`
- Revisado por: `Lejzer Trana`
- Aprobado por: `Lejzer Trana`

## Objetivo
Estandarizar el registro y control de todos los documentos institucionales y operativos de GoBeyond, asegurando trazabilidad, vigencia, propiedad, aprobacion y seguimiento de revision.

## Alcance
Aplica a politicas, SOPs, instructivos, checklists, matrices, formatos y anexos controlados.

## Responsables
- Administrador documental.
- Duenos de proceso.
- Revisores asignados.
- Aprobadores autorizados.

## Instrucciones de uso
1. Registrar una fila por cada documento controlado.
2. Actualizar el formato ante cambios de version, estado o fecha de revision.
3. Conservar el historico de versiones sin eliminar registros previos relevantes.
4. Utilizar este formato como fuente maestra oficial para auditoria documental.
